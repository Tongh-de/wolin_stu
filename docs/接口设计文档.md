# 拾光学子成长管理平台 - 接口设计文档

**版本**: v1.0.0  
**更新日期**: 2025年  
**Base URL**: `http://127.0.0.1:8099`

---

## 一、接口概述

### 1.1 基本信息

| 属性 | 说明 |
|------|------|
| 协议 | HTTP/HTTPS |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |
| API文档 | `/docs` (Swagger UI) |
| 前端界面 | `/static/index.html` |

### 1.2 认证方式

除登录/注册接口外，所有接口都需要在请求头中携带JWT Token：

```
Authorization: Bearer <access_token>
```

### 1.3 通用响应格式

所有接口返回统一的JSON结构：

```json
{
    "code": 200,
    "message": "success",
    "data": {}
}
```

**HTTP状态码规范**：

| 状态码 | 含义 | 使用场景 |
|--------|------|---------|
| 200 | OK | 查询/更新/删除成功 |
| 201 | Created | 创建成功 |
| 400 | Bad Request | 参数错误 |
| 401 | Unauthorized | 未认证/Token过期 |
| 403 | Forbidden | 无权限 |
| 404 | Not Found | 资源不存在 |
| 500 | Internal Server Error | 服务器错误 |

## 二、认证模块 (Auth)

### 2.1 用户注册

**POST** `/auth/register`

**请求体**：
```json
{
    "username": "string",      // 用户名 (3-50字符，字母数字下划线)
    "password": "string",      // 密码 (至少8位，大小写字母+数字)
    "role": "student",         // 角色 (固定为student)
    "student_id": null,        // 学生ID (可选)
    "teacher_id": null         // 教师ID (可选)
}
```

**响应** (200)：
```json
{
    "code": 200,
    "message": "注册成功，请等待管理员审核后登录"
}
```

**错误响应** (400/403)：
```json
{
    "detail": "用户名已存在" | "禁止公开注册管理员或教师账号"
}
```

---

### 2.2 用户登录

**POST** `/auth/login`

**请求体**：
```json
{
    "username": "string",
    "password": "string"
}
```

**响应** (200)：
```json
{
    "code": 200,
    "message": "登录成功",
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "token_type": "bearer",
    "username": "string",
    "role": "admin | teacher | student",
    "student_id": 1,
    "teacher_id": null
}
```

**错误响应** (401)：
```json
{
    "detail": "用户名或密码错误"
}
```

---

### 2.3 退出登录

**POST** `/auth/logout`

**响应** (200)：
```json
{
    "code": 200,
    "message": "退出成功"
}
```

---

### 2.4 获取当前用户信息

**GET** `/auth/me`

**认证**：Required

**响应** (200)：
```json
{
    "code": 200,
    "data": {
        "id": 1,
        "username": "string",
        "role": "string",
        "is_active": true,
        "student_id": 1,
        "teacher_id": null
    }
}
```

## 九、智能Agent模块 (Agent)

### 9.1 智能对话（流式）

**POST** `/agent/chat`

**认证**：Required

**请求体**：
```json
{
    "question": "查询Java2301班有多少学生",
    "session_id": "uuid-string",           // 可选，会话ID
    "force_model": "kimi",                  // 可选，强制使用模型
    "context": [],                           // 可选，上下文消息
    "persona": "lindaidai",                 // 可选，人格角色
    "user_id": null
}
```

**人格角色**：
- `lindaidai`: 林黛玉
- `xueshuzhuoyou`: 学业卓友
- `psychology_student`: 学生心理疏导
- `psychology_teacher`: 教师心理工作

**响应**：Server-Sent Events (SSE) 流式响应

```
data: {"type": "meta", "session_id": "...", "model_used": "...", "intent": "nl2sql", ...}

data: {"type": "answer", "content": "Java2301班有30名学生"}

data: {"type": "done", "total_time_ms": 1234.56}
```
